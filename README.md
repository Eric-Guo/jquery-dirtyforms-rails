jQuery-DirtyForms-Rails [![Gem Version][version-badge]][rubygems]
=======================

## Installation

Add this line to your application's Gemfile:

    gem 'jquery-dirtyforms-rails'

And then execute:

    $ bundle

See [official readme](https://github.com/snikch/jquery.dirtyforms#readme)
